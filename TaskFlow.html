Conectar Supabase
Supabase es 100% gratuito. Sigue estos pasos — menos de 5 minutos.
1
Ve a supabase.com → Start for free → regístrate con Google. Luego New project, ponle nombre, crea contraseña, elige región → Create project.
2
Menú izquierdo → SQL Editor → New query. Copia y ejecuta este SQL:
create table if not exists tickets ( doc_id uuid default gen_random_uuid() primary key, id text, title text, description text, assignee text, priority text default 'Media', status text default 'Pendiente', creator text, created_by_name text, category text, due date, created_at timestamptz default now() ); create table if not exists team_config ( role text primary key, email text ); alter publication supabase_realtime add table tickets; alter table tickets disable row level security; alter table team_config disable row level security; create table if not exists comments ( id uuid default gen_random_uuid() primary key, ticket_id text, author text, body text, mentions text[], created_at timestamptz default now() ); alter table comments disable row level security; alter table comments add column if not exists reply_to text; create table if not exists mention_tasks ( id text primary key, ticket_id text, comment_id text, mentioned_user text, mentioned_by text, created_at timestamptz default now(), resolved_at timestamptz, response_comment_id text ); alter table mention_tasks disable row level security;
3
Menú izquierdo → ícono de cadena 🔗 "Connect" (arriba) → pestaña API Keys. Copia el Project URL y el Publishable Key.
4
Pégalos abajo y haz clic en Conectar.
